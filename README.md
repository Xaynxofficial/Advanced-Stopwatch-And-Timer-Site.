# Stopwatch-Countdown
Stopwatch/Countdown is an advanced site.

✅ Project Features (Short and Concise)

Stopwatch & Countdown: Start, pause, reset. You can enter minutes for the countdown.
Millisecond Precision: 00:00:00.000 format + human-readable time.
Circular Progression Circle: Seconds loop for stopwatch, percentage of remaining time for countdown.
Lap/Lap System: Add laps, display best lap, total lap count, copy on click.
Persistent Recording: Time, laps, and theme are stored in LocalStorage.
Theming System: Dark/Light toggle.
Keyboard Shortcuts: Space (start/stop), L (lap), R (reset), E (export), F (fullscreen).
Mobile Support: Add laps by double-tapping the circle, fully responsive design.
Fullscreen Mode: Minimal large text display, one-click on/off.
Export: Download all lap recordings as JSON.
PWA Support: Can be installed on your phone and works offline.
Modern Interface: Glass effects, simple design, and strong UI/UX.

✅ Who's it for?

Athletes, runners.
Teachers, those who time their lessons.
Speedrunners and gamers.
Developers and testers.
Anyone who wants a simple yet powerful timer.
